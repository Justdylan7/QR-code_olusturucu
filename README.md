# 🔗 Link → QR Kod Oluşturucu

Modern, hızlı ve kullanıcı dostu bir QR kod oluşturma aracı. Herhangi bir URL'yi anında taranabilir bir QR koduna dönüştürün ve PNG formatında indirin.

## ✨ Özellikler
* **Anlık Oluşturma:** URL girildiği anda QR kod hazır.
* **İndirme Seçeneği:** Oluşturulan QR kodları PNG olarak kaydedebilirsiniz.
* **Modern Tasarım:** Dark mode odaklı, responsive (mobil uyumlu) arayüz.
* **Güvenli:** Tamamen istemci taraflı (client-side) çalışır, verileriniz sunucuya gönderilmez.

## 🛠️ Kullanılan Teknolojiler
* HTML5 & CSS3 (Custom Properties)
* JavaScript (ES6+)
* [QRCode.js](https://davidshimjs.github.io/qrcodejs/) kütüphanesi

## 🚀 Hızlı Başlangıç
Bu projeyi bilgisayarınızda çalıştırmak için:
1. Repoyu klonlayın: `git clone https://github.com/kullaniciadi/repo-adi.git`
2. `index.html` dosyasını tarayıcınızda açın.
